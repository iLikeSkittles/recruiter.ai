# Recruiter.ai - Recruitment with AI

Recruiter.ai is a web application that leverages AI technology to enhance the recruitment process. It allows HR managers and recruiters to generate job descriptions and streamline the hiring process.

## Features

- Easily create job descriptions based on specific prompts and templates.
- AI-powered candidate matching based on job requirements.
- Track and manage candidate applications.
- Collaborate with team members on the hiring process.

## Technologies Used

- HTML
- CSS
- JavaScript
- Express.js
- ChatGPT API
- MongoDB (optional for candidate tracking)

## Setup

1. Clone the repository:

   ```
   git clone <repository_url>
   ```

2. Install the dependencies:

   ```
   cd recruiter-ai
   npm install
   ```

3. Set up the environment variables:

   - Create a `.env` file in the project root.
   - Add the following environment variables:
     - `API_KEY` - Your ChatGPT API key.
     - `MONGODB_URI` (optional) - Your MongoDB connection URI if using candidate tracking.

4. Start the application:

   ```
   npm start
   ```

5. Open your web browser and navigate to `http://localhost:3000` to access the application.
